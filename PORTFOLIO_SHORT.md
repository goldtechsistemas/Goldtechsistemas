# Gold Tech Sistemas - Website

**Descrição Curta para Portfólio:**

Website corporativo para plataforma ERP de varejo, desenvolvido com HTML5, CSS3 e JavaScript vanilla. Apresenta layout responsivo com 5 interfaces sobrepostas, carrosséis infinitos arrastáveis, animações ao scroll e formulário integrado com Formspree.

**Principais destaques:**
- 🎨 Layout complexo responsivo (6+ breakpoints)
- 🎬 Carrossel infinito sem pulos com drag/swipe
- ⚡ Sem dependências externas (0 frameworks)
- 📱 Mobile-first com Flexbox/Grid
- 🎯 Intersection Observer para animações performáticas

**Tecnologias**: HTML5 | CSS3 (Grid, Flexbox, Variáveis) | JavaScript Vanilla | Formspree

**Desafio resolvido**: Implementar carrossel profissional com scroll infinito suave + posicionar 5 mockups sobrepostos responsivamente, tudo sem bibliotecas JavaScript.
